Ändra version av gradle och se till att denna version används i projektet = ./gradlew wrapper --gradle-version 9.2.0

Kompilera = ./gradlew compileJava
Kör klasser = ./gradlew run
Kör tester = ./gradlew test
Generera javadoc = ./gradlew javadoc
Paketering = ./gradlew jar

ELLER 

Kör allt på samma = ./gradlew build 

Detta genererar däremot inte javadoc, vilket vi senare behöver köra manuellt.
